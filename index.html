NC NEWS API

This page contains documentation for the various endpoints API

Available Endpoints

  • GET /api/topics

    Retrieve a list of all topics

    Example response:

    
{
    "topics": [
        {
            "slug": "coding",
            "description": "Code is love, code is life"
        },
        {
            "slug": "football",
            "description": "FOOTIE!"
        },
        {
            "slug": "cooking",
            "description": "Hey good looking, what you got cooking?"
        }
    ]
}

  • GET /api/users

    Retrieve a list of all users

    Example response:

    
{
    "users": [
        {
            "username": "John",
            "name": "John Smith",
            "avatar_url": "https://vignette.wikia.nocookie.net/mrmen/images/d/d6/Mr-Tickle-9a.png/revision/latest?cb=20180127221953",
        },
        {
            "username": "grumpy19",
            "name": "Paul Grump",
            "avatar_url": "https://vignette.wikia.nocookie.net/mrmen/images/7/78/Mr-Grumpy-3A.PNG/revision/latest?cb=20170707233013",
        },
    ]
}

  • GET /api/articles

    Retrieve a list of all articles

    Example response:

    
{
    "articles": [
        {
            "article_id": 34,
            "title": "The Notorious MSG's Unlikely Formula For Success",
            "topic": "cooking",
            "author": "grumpy19",
            "created_at": "2020-11-22T11:13:00.000Z",
            "votes": 0,
            "article_img_url": "https://images.pexels.com/photos/2403392/pexels-photo-2403392.jpeg?w=700&h=700",
            "comment_count": 11
        },
        {
            "article_id": 12,
            "title": "The battle for Node.js security has only begun",
            "topic": "coding",
            "author": "tickle122",
            "created_at": "2020-11-15T13:25:00.000Z",
            "votes": 0,
            "article_img_url": "https://images.pexels.com/photos/10845119/pexels-photo-10845119.jpeg?w=700&h=700",
            "comment_count": 7
        },
    ]
}

  • GET /api/articles/:article_id

    Retrieve a single article object

    Example response:

    
{
    "article": {
        "article_id": 1,
        "title": "How to use this API",
        "topic": "coding",
        "author": "Bob",
        "body": "Culpa corporis earum et earum officia a est atque at. Quidem quo recusandae delectus autem possimus blanditiis optio. Sed culpa culpa.",
        "created_at": "2020-11-07T06:03:00.000Z",
        "votes": 0,
        "article_img_url": "https://images.pexels.com/photos/11035380/pexels-photo-11035380.jpeg?w=700&h=700",
        "comment_count": 8
    }
}

  • PATCH /api/articles/:article_id

    Update an article

    No data is returned

  • GET /api/articles/:article_id/comments

    Retrieves a list of all comments for a given article

    Example response

    
{
    "comments": [
        {
            "comment_id": 89,
            "body": "Esse et expedita harum non. Voluptatibus commodi voluptatem. Minima velit suscipit numquam ea. Id vitae debitis aut incidunt odio quo quam possimus ipsum.",
            "article_id": 1,
            "author": "John",
            "votes": 2,
            "created_at": "2020-10-24T07:08:00.000Z"
        },
        {
            "comment_id": 86,
            "body": "Et explicabo dignissimos officia dolore rerum aliquam corrupti. Culpa corporis earum et earum officia a est atque at. Quidem quo recusandae delectus autem possimus blanditiis optio. Sed culpa culpa. Exercitationem nemo aspernatur alias ut qui.",
            "article_id": 1,
            "author": "Bob",
            "votes": 14,
            "created_at": "2020-10-04T02:03:00.000Z"
        },
    ]
}

  • POST /api/articles/:article_id/comments

    Post a comment on a given article and retrieve the newly created comment object

    Only accepts data in the following format

    
{
          username: "butter_bridge",
          body: "This is a test comment",
}

    Example response

    
{
	"comment": {
		"comment_id": 200,
		"article_id": 1,
		"body": "This is a test comment",
		"votes": 0,
		"author": "user1",
		"created_at": "2026-02-10T13:58:22.952Z"
	}
}

  • DELETE /api/comments/:comment_id

    Delete a comment and retrieve the newly created comment
